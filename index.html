PubMed Knowledge Graph

The latest PKG version is now available on the figshare.

And the data in SQL format can be obtained from the cloud disk of Sun Yat-sen University, including tables beginning with A, B, and C. PKG will take up a lot of disk space, so it is recommended to click on the folder name "PKG2023S4" and select the subset you need.
